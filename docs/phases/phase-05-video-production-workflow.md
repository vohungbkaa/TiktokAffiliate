# Phase 05: Video Production Workflow

## Muc Tieu

Ho tro san xuat video nhanh, ban dau uu tien asset package cho CapCut thay vi render tu dong hoan toan.

## Level 1: Manual-Assisted

- Generate script.
- Generate voice-over.
- Generate subtitle.
- Export shot list.
- Export checklist edit CapCut.

## Level 2: Semi-Automated

- Upload anh/video san pham.
- Chon template 9:16.
- Generate subtitle file.
- Render bang FFmpeg neu can.
- Luu final video URL.

## Tasks

- Tao model `Video`.
- Tao asset package export.
- Tao voice-over generation.
- Tao subtitle generation.
- Tao checklist edit.
- Tao upload asset.
- Tao video status flow.

## Definition Of Done

- Mot content idea export duoc thanh goi asset de edit video.
- Luu duoc video draft/final.
- Giam thoi gian tao 1 video xuong con 10-20 phut.

## Khong Lam Trong Phase Nay

- Chua auto post TikTok.
- Chua render hang tram video.
- Chua reup video cua creator khac.
